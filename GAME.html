Shuffle Game

How it works: 1. The HTML creates a simple page with a result display, play button, and reset button. 2. The CSS styles the page. 3. The JavaScript: - Creates an array of numbers from 1 to 20. - Defines two functions: shuffleGame and resetGame. - Attaches event listeners to the play and reset buttons. 4. When the play button is clicked: - The shuffleGame function shuffles the numbers array. - Selects a random number from the shuffled array. - Displays the result. - Hides the play button and shows the reset button. 5. When the reset button is clicked: - The resetGame function resets the numbers array. - Clears the result display. - Shows the play button and hides the reset button. Shuffle Game
